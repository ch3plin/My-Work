# 📊 Statistics Project with Python

## 🔍 Overview
This project applies statistical methods using Python to analyze datasets and extract meaningful insights.  
It demonstrates practical applications of descriptive statistics, probability distributions, hypothesis testing, and data visualization.

## 🛠️ Tools & Technologies
- Python (NumPy, Pandas, SciPy, Matplotlib, Seaborn)
- Jupyter Notebook / VS Code
- Git & GitHub for version control
